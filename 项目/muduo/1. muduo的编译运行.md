# muduo的编译运行

## 1.安装依赖库

```bash
# 安装g++
sudo apt install g++ 
# 安装cmake
sudo apt-get install cmake  make 

# 安装boost
sudo apt-get install libboost-dev libboost-test-dev

# 三个非必须的依赖库:curl、c-ares DNS、Google Protobuf  
sudo apt-get install libcurl4-openssl-dev libc-ares-dev
sudo apt-get install protobuf-compiler libprotobuf-dev
```

## 2.编译

```bash
/* 编译muduo库和它自带的例子，生成的可执行文件和静态库文件位于
 build/release-cpp11/lib */

./build.sh -j4

// 将muduo头文件和库文件安装到build/release-install-cpp11/lib
./build.sh install 
```

## 3.运行

```bash
cd  build/release-cpp11/bin

./inspector_test  //创建了一个http服务端

打开浏览器访问IP和端口即可 // 需先查看本机的ip
```
